# Benchmark Reports

- **[Production Solver Benchmark](benchmark_results_production.md)**: Full solver suite report generated by `benchmarks/run_benchmarks.py`.
- **[Spectral Convergence Benchmark](spectral_convergence.md)**: Worst-case PE-Quad vs Newton-Schulz spectral contraction analysis from `benchmarks/spectral_convergence.py`.

## Reproduce

```bash
uv run python benchmarks/run_benchmarks.py --run-name production_fullsuite --integrity-checksums
uv run python -m benchmarks.spectral_convergence --run-name spectral_convergence
```

Both benchmark suites write run manifests and reproducibility fingerprints, and integrity `.sha256` sidecars are enabled by default.
